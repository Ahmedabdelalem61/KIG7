====================
HR UAE Backend Theme
====================

First-party backend theme addon for the HR UAE Admin suite.

Highlights
==========

* Refreshed backend shell with a softer navigation and content chrome
* Theme presets tuned for the HR workspace
* Live user preference switcher in the systray
* Admin defaults through Settings and the HR UAE Configuration menu
* Visual polish for dashboards, forms, lists, kanban cards, search panels and app menus

This addon is intended to live alongside the existing HR UAE modules as a
normal custom addon, not as a third-party theme package.
